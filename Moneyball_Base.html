Calculamos R² y determinamos qué tan bueno es el modelo. ***¶

Necesitamos una mtrica para saber que tan bueno es nuestro modelo.

Para eso usamos el coeficiente de determinacin R.

R calcula que tan relacionado est nuestro modelo con los datos o que tanto nuestros datos son explicados por nuestro modelo.

La forma en que se hace es comparando los resultados con la media, bsicamente estamos diciendo: si tu modelo es peor para predecir que la media de los datos, tu modelo es muy malo.

El valor es 0 si es igual de bueno para predecir que la media, usa nmeros negativos para indicar que es peor y nmeros positivos para indicar que es mejor, el valor mximo es 1.

Por lo tanto queremos que nuestro coeficiente sea lo ms cercano a 1.

Este resultado nos dice que el presupuesto de un equipo si influye en los juegos ganados, pero no de manera contundente. Lo cul es algo contraintuitivo!

Regresión Lineal Múltiple¶

Cmo un equipo hace carreras?

1.- porcentaje de veces que un jugador llega a base (PBase). Como sea que eso pase, base por bola, al batear, como sea.

2.- porcentaje de bases alcanzadas por un jugador en cada batazo (SLG). Osea cada que el jugador le da a la bola le cuentas cuntas bases avanza y sacas un promedio de todas las veces que esto pasa, dndole mayor peso a las bases mas altas.

3.- Hay una variable que la mayora de los equipos toman en cuenta: el porcentaje de veces que el jugador llega a base al betear (PBat). as que la tomaremos en cuenta tambin.

El siguiente anlisis se llama Regresin lineal mltiple, ya que estamos comparando ms de una variable con otra.

Y podemos usar este mtodo para encontrar qu variables son ms significativas para predecir DC.

Variables significativas¶

Qu variables nos hacen tener ms carreras a favor?

Lo primero que podemos notar es que le coeficiente de determinacin es muy bueno, tenemos un buen modelo.

Podemos ver tambin, que Porcentaje de Bateo tiene un coeficiente negativo, esto nos est diciendo que mientras mayor sea PBat menor es nuestra estimacin de nmero de carreras, lo cual no tiene sentido.

Multicolinealidad!¶

Este problema es causado por algo que llamamos Multicolinealidad: Lo que est pasando en nuestro anlisis es: la variable PBat, est aportando informacin repetida o que ya inferimos de una de las otras dos variables por lo tanto al momento de ajustar nuestra lnea el modelo debe disminuir un poco esa opinin repetida para tener un modelo ms balanceado y eso causa el coeficiente negativo.
